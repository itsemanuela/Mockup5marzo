Come creare un template per i post

di Mario Rossi - pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logici la pagina, identificando le parti comuni da applicare a tutte le pagine.

Le varie pagine saranno quindi suddivise in:

  • Contenuti di intestazione del tag <header>
  • Contenuti centrali variabili del tag <main>
  • Contenuti di fine pagina comuni nel tag <footer>

Formattazione del codice

Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con i CSS.

  1. Progettare su carta il markup finale
  2. Scrivere l'HTML dai tag esterni a quelli interni
  3. Applicare gli stili CSS

L'applicazione degli stili CSS è cresciuta negli anni fino a raggiungere una diffusione pressoché totale.
